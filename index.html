Pear

Credit where credit is due.

Pear is a command line interface to help people add "Co-Authored-By:" trailers to their git commit messages. It's goals are to make it easier to:

  • use git as normal while pairing - no special commands to learn
  • map GitHub usernames to emails automagically for you
  • allow pairing to be set a per-project basis

Install

The best way to install is `brew install pear`.

Setup

Once installed, start by doing `pear init` - this will do a few things for you:

  • create a `~/.pear-authors.json` file
  • globally ignore `.pear` files
